# OneHeart SQL & Database Layer

**ส่วนประกอบของระบบเก็บข้อมูล OneHeart**

## Structure

- **Proof Layer**: บันทึกการทำงาน + หลักฐาน (ภาพ/เสียง/GPS)
- **Action Engine**: เก็บผลการกระทำและ Impact Unit
- **Impact Economy**: ตาราคำนวณ IU, Sponsor Token, Shared Pool
- **Quest System**: ฐานข้อมูลเควส + Skill Tree
- **Nation Dashboard**: Real-time analytics ประเทศ
